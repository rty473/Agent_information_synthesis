# What is Power Automate? — Flow Types Overview

Power Automate is a no-code/low-code automation platform with drag-and-drop workflow building, hundreds of pre-built connectors (Microsoft 365, Salesforce, SharePoint, Teams), native Copilot AI integration, environment isolation, role-based access, and audit logging. Requires a Microsoft work or school account.

**Three flow types:**

- **Cloud Flows**: Run in the cloud, trigger via event (automated), manual (instant), or time schedule (scheduled). Ideal for cross-application tasks like notifications, data sync, approvals, and document workflows.
- **Desktop Flows (RPA)**: Automate desktop apps, legacy systems, and web browsers without APIs using Power Automate for Desktop. Supports recorded actions and drag-and-drop building.
- **Generative Actions (Preview)**: Specify only the *intent*; AI determines the correct sequence of actions. Enables adaptive automation that handles variability without explicitly defining each step.

Flows can also be created and managed within Power Apps via the Power Automate pane in Power Apps Studio.
