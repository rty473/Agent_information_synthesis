# Make Product Overview — Automation and AI Platform

Make is a visual automation platform connecting 3,000+ pre-built apps, enabling complex automations without coding. 400,000+ active customers. Free tier available; Enterprise plans via sales.

**Core builder features**: Drag-and-drop scenario canvas; if/else branching; HTTP/Webhooks for API and external triggers; error routes; scheduled and real-time execution; Make Code (JavaScript or Python execution; Enterprise supports third-party libraries); Notes for in-scenario documentation.

**AI capabilities:**
- *Maia by Make*: Integrated AI assistant for natural language scenario creation and debugging.
- *AI Agents*: Reusable, adaptive agents operating within the visual canvas with real-time reasoning visualization. Built across 3,000+ app integrations via natural language.
- *AI Mapping*: Automates data transformation between applications using AI inference.
- *Make AI Tools*: Prebuilt categorization and summarization modules requiring no external model setup.
- *Make AI Provider*: Native LLM access without external API keys.
- *MCP Server & Client*: Exposes Make scenarios as callable tools for external AI agents via Model Context Protocol; also enables Make to consume external MCP tools.
- *300+ pre-built GenAI apps* covering translation, image processing, content generation, classification.

**Enterprise features:**
- *Make Grid*: Visual map of the entire automation landscape — 40+ app connectors, real-time credit and data monitoring, automated issue highlighting, snapshot sharing.
- Security: GDPR, SOC 2 Type II, encryption in transit and at rest, SSO, Authenticated Webhooks, Credits per Teams (team-level resource allocation).

**Competitive positioning**: Explicitly positioned against Zapier, n8n, Workato, Tray.io. Key differentiators: visual-first interface, native AI agents within the scenario canvas, Make Grid for org-level visibility, and deterministic automation + adaptive agentic AI in a single platform.

Platform ratings: Capterra 4.8/5, G2 4.7/5, GetApp 4.8/5, Gartner Peer Insights 4.6/5.
