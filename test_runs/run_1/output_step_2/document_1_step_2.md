# Explore the Power Automate Home Page

Power Automate enables automated workflows between apps and services for tasks like file sync, notifications, data collection, and approvals.

**Key navigation items:** Home, Create, Templates, Learn, My Flows, Approvals, Solutions, Process Mining, AI Models, Desktop Flow Activity. Items can be pinned/unpinned from the left nav pane.

**Environments** create boundaries between work contexts (e.g., dev vs. production). Roles determine access: Admin, Maker with data access, Maker without full data access, and Run only user. Flows created in one environment are not visible in another; components cannot be easily moved between environments.

**AI Copilot** (org-level opt-in): Available on the home screen when AI is enabled. Users describe a workflow in natural language and Copilot generates the automation structure.

**Contextual help**: Available during flow building via the Microsoft Copilot Studio bot (Power Platform virtual agent).
