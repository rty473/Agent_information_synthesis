# Make Product Overview

## What Make Does

Make is a visual automation platform that enables users to "Build workflows and AI agents on a visual automation platform." The platform connects over 3,000 pre-built applications and allows users to create complex automations without coding expertise. Its tagline positions it as enabling "Idea to execution in hours not weeks."

## Core Features

### Visual Builder and Development
- Drag-and-drop interface for workflow creation
- **Maia by Make**: "Build and troubleshoot with natural language. Faster results, fewer errors." — an integrated AI assistant for scenario creation and debugging
- Support for custom APIs and proprietary system integrations via HTTP/Webhooks
- **Make Code**: Custom JavaScript or Python execution within scenarios (Enterprise tier supports third-party libraries)
- **Notes**: In-scenario documentation for team collaboration

### Workflow Capabilities
- Flow control with conditional statements (if/else branching)
- Data manipulation using drag-and-drop functions
- HTTP/Webhooks for API requests and external triggers
- Error handling with dedicated error routes
- Scheduled and real-time execution modes

### Operational Insights
- Analytics Dashboard for real-time workflow performance tracking
- Observability tools to monitor automation utilization and data consumption
- Role-based access control (RBAC) for team and permission management
- Audit logging for compliance and governance

## AI Capabilities

Make positions AI as a core platform pillar rather than an add-on feature.

### AI Agents
- Reusable, intelligent agents that "adapt in real time" to business needs
- Agent orchestration across 3,000+ applications
- Natural language-based agent creation
- Agents operate within the visual scenario canvas with real-time visualization of reasoning steps

### AI Tools and Features
- **300+ pre-built GenAI applications**: Cover tasks like translation, image processing, content generation, and classification
- **AI Mapping**: Automates data transformation between applications using AI inference
- **Make AI Tools**: Prebuilt modules for categorization and summarization without external model configuration
- **Maia AI assistant**: Natural language workflow building and troubleshooting
- **MCP Server & Client**: Secure agent access to external data and tools via the Model Context Protocol — enables Make scenarios to be called as tools by external AI agents

### Make AI Provider
Native large language model (LLM) access without requiring external API management or separate API keys — users can access AI capabilities directly within Make's platform.

## Integration Library

- **3,000+ pre-built app connectors**: Covers the full spectrum of business software
- Featured integrations include: OpenAI (ChatGPT, Sora, Whisper), Anthropic Claude, HubSpot CRM, Salesforce, monday.com, NetSuite, Slack, Canva, DeepSeek AI, Perplexity AI, Google Workspace, Airtable, Notion

## Enterprise Features

### Make Grid
A unique visual orchestration platform providing "an instant, interactive map of your agentic automation landscape":
- Supports 40+ app connectors
- Real-time credit and data consumption monitoring
- Automated issue highlighting
- Snapshot sharing for cross-team collaboration

### Security and Compliance
- GDPR compliance
- SOC 2 Type II certification
- Encryption protocols (in transit and at rest)
- Single Sign-On (SSO)
- Credits per Teams (Enterprise): resource allocation control at team level
- Authenticated Webhooks: systemic webhook security

## Use Cases and Solutions

Make serves multiple business functions:
- Marketing automation (social media, email campaigns, content repurposing)
- Sales cycle optimization (lead enrichment, CRM updates, outreach sequences)
- Operations management (ticket routing, approval workflows, data sync)
- Customer experience automation (support deflection, response drafting)
- Finance process automation (invoice processing, expense categorization)
- IT workflow scaling (incident management, provisioning)
- HR process automation (onboarding, document collection)
- Workplace productivity (meeting notes, reporting, notifications)

## Competitive Positioning

Make explicitly positions itself as an alternative to:
- Zapier
- n8n
- Workato
- Tray.io

Key differentiators cited: visual-first approach, native AI agent support within the scenario canvas, Make Grid for organizational visibility, and the combination of deterministic automation with adaptive agentic AI in a single platform.

## Adoption and Pricing

- **400,000+ active customers**
- Free tier available with immediate signup
- Paid plans with increasing operations limits and features
- Enterprise plans available (contact sales)
- Review platform ratings: Capterra (4.8/5), G2 (4.7/5), GetApp (4.8/5), Gartner Peer Insights (4.6/5)

## Platform Philosophy

Make emphasizes "citizen development" — enabling non-developers across an organization to build production-grade automations. The visual-first approach is designed to make automation accessible while maintaining the depth required for enterprise-scale deployments. The combination of traditional workflow automation with modern agentic AI is presented as the platform's primary strategic differentiation.
