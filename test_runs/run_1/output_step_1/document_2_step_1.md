# What is Power Automate?

With its automation capabilities, Power Automate helps you streamline your business processes and automate repetitive tasks. Its intuitive interface and many connectors allow you to create workflows with little to no knowledge of coding. You can drag and drop components and set up workflows to save time and improve efficiency. Power Automate can handle simple tasks like sending notifications and more complex processes across multiple apps and services. It's flexible and scalable, making it useful for various automation needs in a modern workplace.

Before you can create a workflow, you need to sign up for Power Automate with a Microsoft work or school email address.

## Types of Flows

Power Automate provides three types of flows to meet different automation needs. Each type of flow is designed for specific scenarios and can be used independently or in combination with others.

### Cloud Flows

Create a cloud flow when you want your automation to be triggered either automatically, instantly, or via a schedule.

Cloud flows run in the cloud and can connect to hundreds of services via connectors. They are ideal for automating tasks that span multiple applications and services, such as:
- Sending notifications when specific events occur
- Syncing data between systems
- Processing approvals
- Automating document workflows

Cloud flows support three trigger models:
- **Automated**: Triggered by an event (e.g., receiving an email, a new row in a database)
- **Instant**: Manually triggered by a user on demand
- **Scheduled**: Run on a recurring time schedule (e.g., daily, weekly)

### Desktop Flows

Use desktop flows to automate tasks on the web or the desktop. Desktop flows (also known as Robotic Process Automation or RPA) enable automation of legacy applications, desktop software, and web browsers that don't have APIs.

Desktop flows use Power Automate for Desktop, which allows users to record actions and replay them, or build flows using a drag-and-drop interface. This is particularly useful for:
- Automating repetitive data entry
- Extracting data from older systems without APIs
- Automating browser-based workflows

### Generative Actions (Preview)

Use generative actions in your flow if you want to specify only the *intent* of the action and then have the AI choose the right set of actions in the right order based on your input, context, and intent.

This represents the most advanced form of Power Automate automation — rather than explicitly defining each step, you describe the goal and the AI determines the execution path. This enables more flexible, adaptive automation that can handle variability in inputs and processes.

## Create and Manage Flows in Power Apps

Your Power Automate license gives you rights to create and manage flows in Power Apps. After signing in to Power Apps, you can select **Flows** on the left navigation pane. You can also create and manage flows through the Power Automate pane within Power Apps Studio, enabling tight integration between app development and automation.

## Key Capabilities

- **No-code/low-code interface**: Drag-and-drop workflow builder accessible to non-developers
- **Connector ecosystem**: Hundreds of pre-built connectors to popular services (Microsoft 365, Salesforce, SharePoint, Teams, and more)
- **AI integration**: Native Copilot assistance for building and troubleshooting flows
- **Enterprise-grade**: Environment isolation, role-based access, audit logging, and compliance features
- **Cross-platform**: Works with cloud services, desktop applications, and legacy systems
